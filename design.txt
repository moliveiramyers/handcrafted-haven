 1. Color Palette (Warm & Organic Earth Tones)
 Primary Accent: #B45309 
 (Warm Amber/Terracotta) —
  Used for primary buttons, active links, and call-to-actions.
  
Secondary Accent: #065F46
(Deep Forest Green) — 
Used for badges, categories, and verified seller icons.

Backgrounds: #FAFAF9 (Off-White/Stone)
for the main body to feel natural;
#FFFFFF (Pure White) for product cards.

Typography Contrast: #1C1917 (Dark Charcoal)
for headings and main body text to guarantee maximum reading accessibility. 
2. Typography StrategyHeadings (<h1> - <h3>):
Playfair Display or Merriweather (Serif font family). 
Transmits an elegant, artistic, and traditional "handcrafted" identity.

Body & UI Elements: 
Inter or Roboto (Sans-Serif font family). 
Clean, highly legible, and optimized for mobile screens.📐 

3. Layout ArchitectureGrid System: Fully fluid and responsive. 
Mobile screens will display a single-column stack, tablets will scale to 2 columns, and desktops will feature a spacious 3 or 4-column layout for product browsing.

UI Elements: Cards with subtle rounded corners and clean box shadows to elevate product photography. 
   

folder structure:

text📁 app/
├── 📁 (auth)/                # Grupo de rotas para autenticação (invisível na URL)
│   ├── 📁 register/          # Rota: /register
│   │   └── page.jsx          # Form de criar conta
│   └── 📁 login/             # Rota: /login
│       └── page.jsx          # Form de sign in
├── 📁 profile/               # Rota: /profile
│   ├── page.jsx              # Profile page & edit profile
├── 📁 products/              # Rota: /products
│   ├── page.jsx              # List de produtos e filter
│   ├── 📁 create/            # Rota: /products/create
│   │   └── page.jsx          # Form de produto (Create)
│   └── 📁 edit/[id]/         # Rota dinâmica: /products/edit/123
│       └── page.jsx          # Form de editar e apagar produto (Edit/Delete)
├── 📁 components/            # [VIEWS] Componentes reutilizáveis globais
│   ├── navbar.jsx            # Menu de navegação
│   └── footer.jsx            # Rodapé do site
├── 📁 actions/               # [CONTROLLERS] Lógica de negócio e Server Actions
│   ├── auth.js               # Login, logout, registar conta
│   └── products.js           # Criar, editar e apagar produtos
├── 📁 lib/                   # [MODELS] Configurações de dados e conexões
│   └── db.js                 # Conexão com o banco de dados (Prisma/MongoDB/Postgres)
├── layout.js                 # Layout global (injeta o nav e o footer em tudo)
└── page.jsx                  # Página inicial (Home)
